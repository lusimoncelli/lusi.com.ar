🎨 Dibujito del día :)🔗

Dibujito del día is a WebApp with functionalities similar to Microsoft Paint, developed using HTML, CSS, and JavaScript. Every day, it presents the user with a new word as inspiration for drawing.

👓 BARCO App: Augmentative and Alternative Communication🔗

B.A.R.C.O. App is an Android app developed in Java that is compatible with a prototype of glasses equipped with IR sensors to facilitate communication for individuals with motor and speech difficulties. By detecting blinks, it allows the user to input text on a keyboard or select pictograms.

⚕️ Classification of Kidney Pathologies from CT Scans🔗

This projects involves a GUI for classifying kidney pathologies from CT Scans images. Developed in Python, it allows users to load an image, select the kidney region, and then applies a segmentation algorithm that isolates the kidneys and classifies the pathology as stones, tumors, cysts, or no pathology. The Machine Learning classification algorithm used is a Decision Tree trained with medical diagnoses..

🤖 Botlusi🔗

Botlusi (@botlusi) is a Twitter/X bot built in Python and deployed on AWS Lambda. It started by simply responding to mentions using a pre-defined list of responses. Later, it evolved to generate its own posts using GPT-2.5, trained on the tweets of the users it followed.
